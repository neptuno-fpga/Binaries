### Instrucciones

F12 -> Menú OSD
Bloq. Despl. -> VGA/RGB 15khz


Teclas (Player 1):
-------------------
Cursores : Direcciones mando nes
C -> Select 
X -> Start
Espacio -> Botón B nes
Alt izquierdo -> Botón A nes


Joysticks:
----------
Mando Atari:
Botón 1 -> Botón B nes
Botón 2 -> Botón A nes


Mando de megadrive:
Mode -> Select
Start -> Start

A -> Botón B  nes
B -> Botón A  nes
C -> Select
