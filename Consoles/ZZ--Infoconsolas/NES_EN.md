### Instructions

F12 -> OSD Menu
Block Scroll -> VGA / RGB 15khz


Keys (Player 1):
-------------------
Cursors: NES command addresses
C -> Select
X -> Start
Space -> B nes button
Left Alt -> A nes button


Joysticks:
----------
Atari controller:
Button 1 -> Button B nes
Button 2 -> A nes button


Megadrive command:
Mode -> Select
Start -> Start

A -> B nes button
B -> A nes button
C -> Select
